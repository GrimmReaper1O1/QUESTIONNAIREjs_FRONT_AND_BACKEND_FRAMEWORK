# QuestionnaireJS

## Introduction


### QuestionnaireJs is an incomplete framework to utilize a revolving multiple choice questionnaire with added video apendix and mp3 content to learn from. It is not, primarily, a testing system 
but is a means to learn through the use of multiple choice questions. The software is built with JavaScript and the API runs on Node.
### I'm taking this into the opensource relm so anyone who wants to work on this can and can profit from it. There is a lot of work involved though.
### It still needs alot of work but the multiple choice questionnaire page is fairly solid. It also lacks adequate css. 
### A video file can be found on youtube here... https://youtu.be/mYrrhssfBMc
### Otherwise it adds to the database incrementally and stores everything in indexedDB. The API or backend utilizes MariaDB and the opensource software to get that all working is found in the package.JSON files associated with the server.
### The questionnaire can be filled and altered but pretty much everything else needs to be created or needs work. The CSS is terrible looking but then I'm a full stack developer with not much time on my hands. The whole thing is still in need of refactoring.
